Saya akan memodifikasi alur deployment (`deployWizard`) di `bot.js` untuk menambahkan opsi **Custom Domain**.

Berikut adalah rencananya:

1.  **Modifikasi Wizard Deployment**:
    *   Menambahkan langkah baru setelah input "Nama Aplikasi".
    *   Bot akan bertanya: *"Apakah Anda memiliki domain sendiri?"* (Pilihan: Ya / Tidak).
    *   **Jika "Tidak"**: Otomatis menggunakan `nama-app.IP-VPS.nip.io` (seperti sebelumnya).
    *   **Jika "Ya"**: Bot akan meminta user memasukkan nama domainnya (misal: `myapp.com`).

2.  **Validasi & Edukasi**:
    *   Jika user memilih "Ya", bot akan mengingatkan untuk **mengarahkan DNS A Record** domain ke IP VPS terlebih dahulu agar SSL bisa aktif.

3.  **Update Logika Deployment**:
    *   Mengubah konfigurasi Nginx dan Certbot untuk menggunakan domain yang dipilih (baik itu `nip.io` atau custom domain).

Dengan ini, Anda bisa memilih mau pakai domain gratisan (otomatis) atau domain pro (custom) saat deploy.
